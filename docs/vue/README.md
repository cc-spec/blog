## 1. 学过的两大框架的对比
- angular使用TypeScript开发
- Vue体积较angular小
- angular中的API接口较多，学习成本大，更适合于大型应用
- Vue 将模板编译成虚拟 DOM 渲染函数。结合响应系统，Vue 能够智能地计算出最少需要重新渲染多少组件，并把 DOM 操作次数减到最少

## 7. 计算属性
- 计算属性存在缓存，即上一次调用计算属性的getter方法后的数据，缓存发生变化才会重新求值；
- 方法：一旦重新渲染就会立即执行这个函数
